# blog-template

## Setup

- Clone the repository

  ```
  git clone git@github.com:deep-stack/blog-template.git

  cd blog-template
  ```

- Install dependencies

  ```
  yarn
  ```

- Modify the content in [index.md](./index.md)

- Generate the corresponding `index.html` file

  ```
  yarn generate
  ```

## Deploy

- Deploy the app

  ```
  serve public
  ```
